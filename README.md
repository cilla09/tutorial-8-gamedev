# Tutorial 8 Game Development
### Priscilla Natanael Surjanto/2306152153

Menurut saya, updated movement dan particle lari meningkatkan kualitas game. Sebelumnya, player yang tidak diberi polishing apa-apa terasa biasa dan hambar, dan kurang menarik untuk dimainkan. Namun setelah diberi polishing seperti yang disebutkan, game menjadi lebih menarik dan asik untuk dimainkan. Feel bermain jadi lebih oke berkat polishing ini.

Untuk spawn rate, saya melakukan playtesting dengan range spawn rate 1-3, dan yang saya temukan adalah 2-2.5 yang paling balanced bagi saya, tidak terlalu lambat maupun terlalu cepat. Hanya butuh sedikit kesabaran untuk menemukan timing yang tepat, tapi tidak perlu banyak trial and error untuk sampai end goal. Sementara, jika ingin membuat game lebih challenging, spawn rate bisa diatur sekitar 1.5 yang lebih sulit tapi masih doable.
